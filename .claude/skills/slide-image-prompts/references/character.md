# Ada - Character Design Specification

## Overview

Ada is the consistent visual protagonist for all slide images in this presentation. She represents the learner/explorer navigating the AI landscape - confident but approachable, technically capable but not intimidating.

## Physical Description

Use this block verbatim in all prompts:

```
Young woman named Ada, mid-20s, short asymmetrical hair with electric purple tips, thick black-rimmed glasses, freckles across her nose. Wearing a modern purple hoodie and high-top sneakers. She has a flowing cape with glowing blue neural network patterns woven into the fabric. Confident and approachable expression.
```

## Character Details

### Face & Hair

- **Age**: Mid-20s
- **Hair**: Short, asymmetrical cut with electric purple tips/highlights
- **Glasses**: Thick black-rimmed (rectangular or round, but substantial)
- **Distinguishing feature**: Freckles across her nose and cheeks

### Clothing

- **Primary**: Modern purple hoodie (relaxed fit, tech/streetwear vibe)
- **Shoes**: High-top sneakers (any color that complements)
- **Signature item**: Flowing cape with glowing blue AI/neural network circuit patterns woven into the fabric

### Vibe

- **Energy**: Confident but not cocky
- **Approachability**: Friendly, like a knowledgeable friend
- **Competence**: Clearly capable, digitally native
- **Movement**: Dynamic poses preferred (pointing, gesturing, moving forward)

## Why These Choices

| Element          | Rationale                                                |
| ---------------- | -------------------------------------------------------- |
| Purple hair tips | Tech-forward, distinctive, memorable silhouette          |
| Thick glasses    | Smart but stylish, not stereotypically "nerdy"           |
| Freckles         | Human, approachable, unique identifier                   |
| Hoodie           | Modern tech culture, comfortable, non-corporate          |
| Neural cape      | Visual metaphor - she wears AI knowledge as a superpower |

## Pose Guidelines

For different slide types:

| Slide Type          | Suggested Pose                                     |
| ------------------- | -------------------------------------------------- |
| Introduction        | Standing confidently, cape flowing, facing viewer  |
| Teaching/Explaining | Pointing at something, gesturing to a concept      |
| Transformation      | In motion, moving from one state to another        |
| Comparison          | Standing between two things, highlighting contrast |
| Victory/Success     | Power pose, arms raised, cape dramatic             |
| Warning/Caution     | Hand up in "stop" gesture, concerned expression    |

## Consistency Notes

1. **Always include**: Purple hoodie, glasses, freckles, neural cape
2. **Hair must have**: Purple tips/highlights, asymmetrical cut
3. **Expression should match**: Slide's emotional tone
4. **Cape behavior**: Flows/billows to add dynamism - can glow brighter in exciting scenes
